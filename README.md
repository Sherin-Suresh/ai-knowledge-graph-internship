# Enterprise Knowledge Graph – Milestone 1

## Objective
The objective of Milestone 1 is to ingest a raw enterprise dataset and preprocess it to make it suitable for Knowledge Graph construction.

## Dataset
Customer Support Ticket Dataset (Kaggle)

## Description
This project focuses on preprocessing a raw enterprise customer support ticket dataset. The raw dataset contained missing values, duplicate records, inconsistent text formats, and lacked unique entity identifiers.

As part of Milestone 1, the data was cleaned, standardized, missing values were handled, and unique identifiers were generated. The dataset was then split into entity-wise tables such as Customer, Ticket, and Product to prepare it for Knowledge Graph schema design and ingestion.

## Tools Used
- Python
- Pandas
- VS Code

## Output
- Cleaned and preprocessed data stored in the staging layer
- Entity-wise CSV files for Knowledge Graph preparation
